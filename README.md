# PPSSPP for Flatpak

## Development (bleeding-edge/master) builds

This Flatpak repository uses the latest commits from [PPSSPP Emulator](https://github.com/hrydgard/ppsspp), instead of releases (tags).

## Purpose

For intended users who wants to try the latest builds, instead of waiting from either Flathub or from the developer itself to bump the version.

## Disclaimer

DO NOT REPORT THE ISSUE to me. For anything related to the emulator, please report [here](https://github.com/hrydgard/ppsspp/issues). For anything related to the Flatpak, please report [here instead](https://github.com/flathub/org.ppsspp.PPSSPP/issues).
